# ARSIA MONS ARCHIVE — Design Brainstorm

## Three Approaches

### 1. TERMINAL NOIR
A degraded CRT terminal interface — green phosphor text on black, scan lines, command-line navigation. The user types commands to access files. Pure text-based immersion with occasional image reveals.
**Probability:** 0.04

### 2. DOSSIER CARTOGRAPHY
A spatial map-based interface — the colony is rendered as an interactive schematic. Users navigate by clicking sectors, rooms, and corridors. Files are physically located in the world. Navigation IS the fiction.
**Probability:** 0.07

### 3. SIGNAL DECAY
A corrupted intelligence broadcast — the interface presents as a recovered data stream being reconstructed in real-time. Content appears through interference patterns, redacted blocks, and signal noise. The page itself is the artifact — glitching, decaying, revealing. Navigation is non-linear: you follow signal threads that branch and cross-reference.
**Probability:** 0.06

---

## CHOSEN APPROACH: SIGNAL DECAY

### Design Movement
**Brutalist Data Archaeology** — inspired by declassified government documents, corrupted digital forensics, and the aesthetic of information that was never meant to be seen. The interface is not a presentation layer — it IS the recovered data. Every element carries the weight of having survived deletion.

### Core Principles
1. **Information as Terrain** — Content is not displayed, it is excavated. Every piece of lore feels like it was pulled from wreckage.
2. **Signal vs. Noise** — The interface oscillates between clarity and corruption. Clean data exists alongside redacted blocks, corrupted sectors, and unresolved fragments.
3. **Non-linear Discovery** — There is no "correct" reading order. Cross-references create a web. Finding a detail in one dossier recontextualizes another.
4. **The System is Dying** — The interface itself shows signs of damage. Glitch artifacts, data corruption markers, timestamp anomalies. The archive is degrading in real-time.

### Color Philosophy
Not a palette — a signal spectrum. The base is DEEP BLACK (the void of deleted data). Information emerges in:
- **Phosphor Green** (#00ff41) — Active system text, confirmed data
- **Amber Warning** (#ff8c00) — Flagged content, analyst notes
- **Blood Red** (#8b0000) — Classified markers, danger, redactions
- **Cold Steel** (#2a3a4a) — Structural elements, borders, containers
- **Signal White** (#e0e0e0) — High-priority text, headers
- **Corruption Purple** (#4a0080) — Anomalous data, Lazarus references
- **Noxis Blue** (#0066cc) — Corporate data streams

### Layout Paradigm
**Fragmented Broadcast** — The page is not a grid. It's a signal being reconstructed. Content blocks appear at varying widths, some overlapping, some offset. The layout suggests a data stream being decoded — some packets arrived intact, some are corrupted, some are still buffering. Navigation happens through a persistent "signal tracker" — a horizontal frequency bar at the top that shows available data channels (agencies, timeline, equipment, conspiracies). Clicking a frequency tunes the display to that data stream.

### Signature Elements
1. **Redaction Blocks** — Black bars with classification stamps that can be "decrypted" on hover/click, revealing hidden text beneath
2. **Signal Interference** — Periodic CSS glitch effects on transitions, suggesting the archive connection is unstable
3. **Timestamp Anomalies** — Every document carries a timestamp, but some are impossible (dates before colony founding, dates in the future). These are clues.

### Interaction Philosophy
Every interaction should feel like you're operating equipment that wasn't designed for you. Clicks produce slight delays. Hover states reveal hidden data. Scrolling triggers "signal acquisition" animations. The archive resists casual browsing — it rewards patience and attention.

### Animation
- Page transitions: horizontal scan-line wipe (like tuning between frequencies)
- Content reveal: typewriter effect for first-time text display
- Hover on redacted blocks: static/noise animation before reveal
- Navigation: frequency-tuning slide with brief signal noise
- Idle state: subtle CRT flicker on borders, occasional data corruption flash
- Scroll: parallax depth on layered content blocks

### Typography System
- **Headers:** "Share Tech Mono" — monospace, technical, cold
- **Body text:** "IBM Plex Mono" — readable monospace for that terminal authenticity
- **Classification stamps:** "Courier Prime" — typewriter weight, official documents
- **Display/Titles:** Custom CSS text with letter-spacing and text-shadow glow effects
- All text has subtle text-shadow matching its color category for that phosphor-glow effect

### Brand Essence
A classified intelligence archive recovered from a purged Martian government server — for obsessive worldbuilding fans who want to lose themselves in a forgotten universe's darkest secrets.
**Personality:** Paranoid. Meticulous. Haunted.

### Brand Voice
Headlines sound like intercepted transmissions: terse, urgent, incomplete.
- "SIGNAL ACQUIRED — DECRYPTION IN PROGRESS"
- "WARNING: FILE INTEGRITY COMPROMISED — PROCEED WITH CAUTION"
CTAs are system commands: "DECRYPT", "ACCESS", "TUNE FREQUENCY"

### Wordmark & Logo
A fractured broadcast tower icon — three vertical lines of decreasing height with a circular signal wave emanating from the top, partially corrupted (missing segments). Rendered in phosphor green with a subtle glow. The mark suggests transmission, damage, and persistence.

### Signature Brand Color
**Phosphor Green (#00ff41)** — the unmistakable color of a terminal that should have been wiped but wasn't. Every key interaction, every reveal, every moment of discovery pulses in this green.
